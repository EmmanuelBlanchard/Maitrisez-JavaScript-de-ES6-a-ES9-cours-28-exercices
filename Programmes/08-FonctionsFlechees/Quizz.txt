    Qu'est qu'une fonction anonyme ?

    Une fonction qui n'a pas de nom
    Une fonction qui n'utilise pas de paramètre
    Une fonction qui s'écrit sans le mot clef "function"

    Bonne Reponse =>

    Une fonction qui n'a pas de nom

    Qu'est qu'une fonction fléchée ?

    une fonction anonyme qui s'écrit sans le mot clef "function" et qui utilise une flèche à la place
    une fonction qui s'écrit avec le mot clef "function" et une flèche pour définir les paramètres.
    une fonction sans paramètre

    Bonne Reponse =>

    une fonction anonyme qui s'écrit sans le mot clef "function" et qui utilise une flèche à la place

    Quelle écriture n'est pas valide
    
    x => x+1
    ( ) => 4*3
    (x,y) => x * y
    x,y => x * y

    Bonne Reponse =>

    x,y => x * y
